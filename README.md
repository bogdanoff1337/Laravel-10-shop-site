### install project 
1 - git clone project

2 - [install sail into project](https://laravel.com/docs/10.x/sail#main-content)
    also u can up project with laradock - but he is mach more bigger 

3 - build sail  

4 - up sail

5 - cp .env.example .env

6 - key generate

7 - composer install 

8 - composer update 

9 - php artisan migrate 

10 - php artisan db:seed
